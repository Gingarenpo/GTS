# GTS - Ginren Traffic System
Minecraft 1.12.2専用Modとして、Minecraftの世界に交通システムの概念を追加します。

GTS1.0と基本構造は変わりませんが、メモリ管理などで不都合があり非常に重くなる仕様が存在したので、そちらを
修正しつつ、1.0になかった機能を追加し、2.0として提供を行う予定です。

## 注意
GTS1.0のパックを使うことはできますが、1.0で設置したブロックなどは全滅するのでご注意ください。メモリ管理の都合上、パックの持ち方を根本的に変更したのでこれは避けられない変更点です。

置き換えには一切対応していないので、手動ですべて置き換えるか、重いのを覚悟で1.0を継続してお使いください。
